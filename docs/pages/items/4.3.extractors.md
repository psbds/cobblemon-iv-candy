## Description

IV Extractors are specialized tools that harvest perfect Individual Values (IVs) from Pok√©mon and convert them into IV Candies. When used on a Pok√©mon, they extract all stats with maximum IVs (31) and create corresponding candies, while reducing those IVs back to 1.

## Usage

### How to Use IV Extractors

1. **Right-click on a Pok√©mon**: While holding an IV Extractor in your hand
2. **Ownership Verification**: The extractor verifies you own the target Pok√©mon
3. **Compatibility Check**: Confirms the Pok√©mon matches the extractor type
4. **IV Analysis**: Scans for perfect IVs (value of 31)
5. **Extraction Process**: Removes perfect IVs and creates candies
6. **Inventory Update**: Adds created candies to your inventory

### Usage Requirements

| Requirement | Description | Failure Message |
|-------------|-------------|-----------------|
| **üéØ Ownership** | Must own the target Pok√©mon | "You do not own this Pok√©mon." |
| **üîó Compatibility** | Extractor type must match Pok√©mon | Type-specific compatibility message |
| **‚≠ê Perfect IVs** | Pok√©mon must have at least one IV at 31 | "This Pok√©mon has no IVs to extract." |
| **üì¶ Held Item** | Must hold extractor in main hand | N/A |

### Compatibility Rules

#### Elemental Type Extractors
- **Primary Type Match**: Works if Pok√©mon's primary type matches extractor type
- **Secondary Type Match**: Works if Pok√©mon's secondary type matches extractor type
- **Dual-Type Support**: Pok√©mon with matching primary OR secondary type are compatible

#### Special Category Extractors
- **Legendary Extractor**: Only works on Legendary Pok√©mon
- **Mythical Extractor**: Only works on Mythical Pok√©mon  
- **Ultra Beast Extractor**: Only works on Ultra Beast Pok√©mon
- **Paradox Extractor**: Only works on Paradox Pok√©mon

### Extraction Results

#### What Gets Extracted
- **Perfect IVs Only**: Only stats with IV value of 31 are extracted
- **Multiple Stats**: All perfect IVs are extracted in a single use
- **Stat Reduction**: Extracted IVs are reduced to 1 (minimum value)

#### Candy Creation
- **One Candy Per Stat**: Each extracted perfect IV creates one targeted candy
- **Candy Type Inheritance**: Created candies match the extractor's type
- **Immediate Availability**: Candies are added directly to inventory

#### Example Extraction Scenarios

| Before Extraction | After Extraction | Candies Created |
|------------------|------------------|-----------------|
| HP: 31, ATK: 25, DEF: 31 | HP: 1, ATK: 25, DEF: 1 | HP Candy + Defense Candy |
| All stats at 31 | All stats at 1 | 6 Candies (HP, ATK, DEF, SPA, SPD, SPE) |
| No perfect IVs | No change | No extraction possible |
